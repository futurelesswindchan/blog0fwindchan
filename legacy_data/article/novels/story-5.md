<h1 class = 'HolyUnion'>Scene 01</h1>

> 地狱里的救赎，通常长着另一副地狱的模样。

锈城的夜晚依旧湿冷。

后巷深处，风织蜷缩在一个生锈的通风管道下，这里是唯一能勉强躲雨的角落。  
她刚在巷口的水洼里，用冻得发僵的手，尽可能地洗去了身上的血迹。

但那股铁锈与罪恶混合的气味，仿佛已经渗入了骨髓，怎么也洗不掉。

破旧的衣服已经彻底失去了蔽体的功能。  
胸口巨大的裂口，让苍白的皮肤和嶙峋的肋骨暴露在湿冷的空气中。

她只能用双臂紧紧环抱住自己和诗集，像一只折断了翅膀的雏鸟。

地上的阴影里，一滩不起眼的、仿佛是积水的墨色液体，开始轻微地蠕动。  
一个半融化的、由流动的黑暗构成的脑袋，缓缓地从里面探了出来。

<span class="bg-griffon">「哟，小祈酱，清理干净了吗？」</span>

那个声音在她脑海中响起，不再是之前的虚弱或哀求，而是带着一种饱餐后的慵懒和了然于胸的调侃。  
<span class="bg-griffon">「动作还挺利索。<br>需不需要我帮你检查一下，有没有漏掉什么不该留下的痕迹？<br>比如……指纹？」</span>

风织没有理会他，只是将脸埋得更深，仿佛想把自己缩进世界的缝隙里。

<span class="bg-griffon">「别这么不爱说话嘛。我们现在可是『共犯』了，得多交流交流感情。」</span>

它的独角戏，被巷口传来的、杂乱的脚步声和手电筒光束打断了。  
<span class="bg-gray">「……妈的，肯定就在这附近！给我仔细搜！」</span>  
是餐馆老板的声音，充满了愤怒和一丝不易察觉的恐惧。

风织的身体瞬间绷紧了。

<span class="bg-griffon">「终于有观众到啦。」</span>  
语气里满是看好戏的愉悦。  
<span class="bg-griffon">「怎么办呢？是出去跟他们打个招呼，还是继续在这里玩捉迷藏？」</span>

它完全没有要动的意思，只是饶有兴致地看着风织，看着她那双因紧张而微微颤抖的蓝色眼睛。

手电筒的光束扫了过来，即将照亮她藏身的角落。  
就在光束触及的前一刻，风织猛地闭上了眼，怀中的书也滑到了地上。

**滋啦——！**

一道微弱但致命的红深紫色闪电，从天而降，精准地击中了巷口的水洼。  
几声惨叫和重物倒地的声音传来，世界暂时恢复了寂静。

猛禽的脑袋开始更剧烈地融化起来，显然刚才那一下耗尽了它最后的能量。

它的语气却依旧平静，甚至带着一丝笑意。  
<span class="bg-griffon">「看到了吗？这就是『我们』能做到的事。」</span>  
它看向巷口那些倒在地上、正试图爬起来的人影。  
<span class="bg-griffon">「但马上就要和你说拜拜咯，到时候……就轮到你和他们打招呼了哦。」</span>

它没有再提「契约」，也没有催促。  
它只是安静地看着她，像一个最耐心的猎人，等待着猎物自己走进陷阱。

风织看着巷口那些倒在地上、正试图爬起来的人影，又低头看了看融化的鸟头。

她不知道自己是什么心情。

这个讨厌的、吵闹的家伙，是她在这个世界上唯一的「同伴」了。  
如果它消失了……  
自己也会立刻被这些人抓住，落入未知的、更恐怖的深渊。

在最后一瞬间，她不再犹豫，努力伸出了颤抖的手。  
不是去捡那本诗集，而是猛地抓向了那团即将溃散的黑暗。

_周围的世界瞬间褪色、剥离。_  
_一切消失了，取而代之的，是一片无边无际的、齐腰深的灰烬海洋。_  
_天空是破碎的，像一块被砸出无数裂痕的镜面。_

_远处，一座福利院的骨架，在永不熄灭的火焰中燃烧着。_  
_这里是她的地狱。_

_风织就站在这片灰烬之海的中央。_  
_在她面前，一个瘦小的、和她一模一样的银发女孩，正蜷缩在地上，无声地哭泣。_  
_那是过去的她，那个只会发抖、什么都做不了的她。_

_一只巨大的、由阴影和电光构成的猛禽，盘旋在哭泣女孩的上空，用一种冰冷的、审视的目光看着她。_  
_它没有说话，但风织能理解它的意思。_

**_选择吧。_**

_是继续留在这里，和这个哭泣的自己一起，沉沦在这片永恒的灰烬中？_  
_还是……_

_风织迈开了脚步。_  
_她踩着没过膝盖的灰烬，一步一步地，艰难地，走向了那只盘旋的猛禽。_

_「别走……」_

_一个微弱的、带着哭腔的声音从身后传来。_  
_是那个蜷缩在地上的“她”在说话。_

_「求你了……别过去……」_  
_那个女孩抬起头，满是泪痕的脸与风织一模一样，眼神里充满了恐惧和哀求。_  
_「它会吃了你的……我们会死的……」_

_风织的脚步，停顿了一瞬。_  
_她能感觉到那个女孩的恐惧，因为那就是她自己的恐惧。_

_「留下来……」女孩的声音变得更急切_  
_「只要我们留在这里，就不会再受伤了……我们可以一直想着院长，想着玛丽亚……只要我们不出去，他们就永远不会消失……」_

_这是她内心最深处的、最懦弱的渴望——用回忆筑起一座安全的囚笼，永远地逃避下去。_

_风织缓缓地、缓缓地回过头，看了一眼那个哭泣的自己。_  
_她的眼神里，没有愤怒，没有不耐烦，只有一种看透了一切的、深不见底的疲惫。_

_她没有回答。_  
_只是重新转过身，继续走向那只沉默的、盘旋的猛禽。_

_她的沉默，就是最响亮的回答。_  
_她的背影，就是最决绝的告别。_

_身后的哭声和哀求声渐渐变小。_  
_当风织走到猛禽的正下方时，她能感觉到，身后那个属于“过去”的自己，已经像被风吹散的灰烬一样，彻底消失了。_

_她亲手杀死了自己的软弱。_  
_不再是之前的麻木或崩溃，而是一种前所未-有-的、混杂着决绝与疲惫的平静。_

_她走到了猛禽的下方，抬起头，迎向那双非人的、琥珀色的眼睛。_  
_她伸出了手。_

_猛禽发出一声无声的、仿佛来自亘古的鸣叫，俯冲而下。_  
_它没有攻击她，而是化作一股纯粹的、活着的黑色墨流，将她完全包裹。_

当她抓住那团冰冷的阴影时，一股前所未有的剧痛和力量洪流，从掌心猛地灌入她的身体。  
她能清晰地「看」到，无数黑色的、带着电光的墨迹，顺着她的手臂倒流而上，最终汇聚在她胸口那片滚烫的皮肤上。

她忍不住发出一声压抑的闷哼，跪倒在地。

当她再次抬起头时，她低头看向自己的胸口。
透过那破烂的衣物，她第一次清晰地看到了它的全貌——

一幅巨大、繁复、由无数盘旋的羽翼和猛禽图腾构成的黑色纹身，从她的左侧锁骨一直延伸到心口。  
它不像死物，更像活物，每一根线条都仿佛在缓缓呼吸，散发着不祥而强大的气息。

脑海里，它的声音再次响起。

<span class="bg-griffon">「欢迎回来。」</span>

这一次，不再是半融化的虚影，而是如同雷鸣般清晰、充满了力量和满足感。

<span class="bg-griffon">「现在，新手教程第一课。伸出你的右手。」</span>

风织的身体，在一种半推半就的本能驱使下，抬起了右臂。  
她的动作还有些僵硬，像是第一次操控不属于自己的肢体。

<span class="bg-griffon">「喊我的名字……哦，我还没告诉你。<br>叫我『格里芬』就行。或者随便什么，『嘿，那只蠢鸟』也行。」</span>

风织的嘴唇动了动，但什么声音也没发出来。

<span class="bg-griffon">「咦，哑巴了？心里想也一样！快点！」</span>

风织闭上眼，在心里默念着那个名字。  
大片的黑色，站在她的手臂上迅速凝聚成形。

**唰！**

格里芬再次出现。  
这一次，它的身形比之前凝实了数倍，羽翼边缘闪烁着危险的电光。  
它发出一声响亮的、充满挑衅意味的鸣叫，飞起来盘旋在半空中。

风织下意识地低头看了一眼自己的胸口。  
那里，皮肤光洁如初，仿佛那幅巨大而狰狞的纹身从未存在过。  
所有的「墨水」，都已汇聚成这只振翅的魔物。

<span class="bg-gray">「怪物……是怪物！」</span>  
刚爬起来的那群人被眼前这一幕吓破了胆，转身就想跑。

<span class="bg-griffon">「B↑~B→~Q↓ Time！——」</span>  
格里芬发出一阵快意的尖笑，双翼一展，数道红紫色的的闪电如暴雨般落下。

小巷重归寂静。

<h1 class = 'HolyUnion'>Scene 02</h1>

> 要重塑一个破碎的灵魂，有时得从修补一副躯壳开始。

格里芬抓着风织的肩膀，穿过冰冷的雨幕，最终将她带到了这座废弃钟楼的顶端。  
这里干燥、僻静，巨大的齿轮和蒙尘的钟摆像远古巨兽的骨骸，在角落里沉默着。

风织一落地，就挣脱了它的利爪，蜷缩到一个相对干净的角落。  
她依然用双臂死死环抱住自己，像一只刚从水里捞出来的、冻得瑟瑟发抖的猫。

格里芬在她头顶盘旋了一圈，落在一根横梁上，居高临下地审视着她。

<span class="bg-griffon">「啊，我亲爱的小朋友，看看你这副模样。」</span>

它的声音，用一种极其夸张的、仿佛在演舞台剧的咏叹调，在空旷的钟楼里回响。

<span class="bg-griffon">「湿漉漉的头发，楚楚可怜的眼神，还有这身……<br>嗯，充满后现代解构主义风格的破布。<br>真是让我这颗不存在的心，都感到了一丝丝的……不忍直视。」</span>

风织没有反应，只是把头埋得更深。

<span class="bg-griffon">「不行，绝对不行！」</span>  
它猛地振翅，带起一阵灰尘，  
<span class="bg-griffon">「作为本大人钦定的合作伙伴，你的审美不能如此堕落！这严重影响了我的格调！」</span>

它绕着风织飞了一圈，最后停在她面前，用翅膀尖煞有介事地指着她。

<span class="bg-griffon">「所以！为了庆祝我们这伟大合作的正式开端，也为了拯救你那岌岌可危的品味<br>——本大人决定，赐予你一份能配得上我的礼物！」</span>  
<span class="bg-griffon">「在此稍候，不要眨眼！最懂时尚的恶魔快递员，马上就回来！」</span>

说完，它不等风织做出任何反应，  
便化作一道迅捷的黑影，从钟楼破碎的彩色玻璃窗中爆射而出，消失在雨夜里。

世界重归寂静。

只剩下风织一人，和窗外永不停歇的、沙沙的雨声。  
她不知道那只蠢鸟在发什么疯，也懒得去想。  
她只是静静地蜷缩着，感受着这份来之不易的、暂时的安宁。

不知过了多久，那道黑影再次从窗外射入。  
伴随着一个高级服装店的纸袋，「啪」的一声，被丢在了她面前。

<span class="bg-griffon">「当当当当~！你的专属造型师回来啦！」</span>

风织迟疑地伸出手，打开了那个印着烫金Logo的纸袋。  
最先映入眼帘的，是一套叠得整整齐齐的、带着蕾丝花边的……贴身衣物。

风织的动作，僵住了。
她那张麻木了的、仿佛已经忘记了如何做出表情的脸上，  
第一次浮现出了一丝极其细微的、混合着难以置信和不知所措的……

红晕。

<span class="bg-griffon">「怎么了？快换上啊。」</span>  
格里芬的声音里充满了期待，  
<span class="bg-griffon">「这可是我精心挑选的，无论是材质还是设计，都堪称艺术品！」</span>

风织猛地将袋子合上，抱在怀里，然后背过身去，面朝墙壁。

<span class="bg-griffon">「喂喂，害羞什么？你的身体哪个部分我没『住』过？」</span>

尽管声音喋喋不休，风织还是用最快的速度，笨拙地换上了那套崭新的衣物。  
布料柔软的触感，和之前那身粗糙的破布截然不同，让她产生了一种奇异的陌生感。

她刚换好，格里芬就迫不及待地绕到她面前，  
用一种产品经理审视样品的挑剔目光，上上下下地打量着她。

然后，它发出一声极其失望的「啧」。  
<span class="bg-griffon">「尺寸好像不太对啊。」</span>

它用翅膀尖戳了戳自己的下巴（如果它有的话），  
<span class="bg-griffon">「完全撑不起来嘛，一点都不饱满。<br>喂，你这几年到底有没有好好吃饭？这发育状况，严重不符合本大人的投资预期啊！」</span>

羞耻感，混合着一丝被冒犯的怒意，瞬间涌了上来。  
风织抓起地上那团湿透了的破布，用尽全身力气朝它丢了过去。  
黑影轻松地一闪，躲开了攻击。

<span class="bg-griffon">「啧啧，恼羞成怒了。」</span>  
它嘴上继续嘲笑着，但却嘀咕了一句，

_真是麻烦得要死……_

<span class="bg-griffon">「等着！本大人去给你申请售后服务！<br>不合身包退换，这也是我们经商的基本素养！」</span>

黑影再次飞了出去。  
这一次，风织坐在原地，抱着膝盖，  
第一次有了一种想把某个东西揪出来打一顿的冲动。

没过多久，格里芬又回来了，丢给她一套明显更合身、款式也更简洁的款式。

在它的监工和吐槽之下，风织终于换好了所有的衣物，  
包括那件干净的白衬衫和笔挺的黑裤子。

她站起身，走到一块巨大的、破碎的玻璃窗前，看着里面那个模糊的、既熟悉又陌生的倒影。  
干练，整洁，像一个……**『正常人』**。

<span class="bg-griffon">「哼，总算是从阴沟里的老鼠，升级成一个勉强能上桌的……嗯，小女仆了。」</span>  
格里芬落在她的肩膀上，用一种「孺子可教」的语气总结道。

<span class="bg-griffon">「感觉怎么样？嗯？」</span>  
<span class="bg-wind">「好一点。」</span>  
<span class="bg-griffon">「好一点？！你说好一点？！！<br>你刚刚说『好一点』是吗？」</span>

...

在这场荒诞的闹剧之后，积攒了数月之久的疲惫感，终于如潮水般袭来。  
风织抱着书，靠着墙壁缓缓坐下，眼皮变得无比沉重。

<span class="text-deep-orange">为我为你为所有的人哭泣，</span>  
<span class="text-deep-orange">当他还是一个小小的孩提</span>  
<span class="text-deep-orange">他的形象你总看得见，</span>  
<span class="text-deep-orange">那俯向你微笑的神圣的脸。</span>

在意识陷入黑暗的前一刻，她听到格里芬在她耳边：  
<span class="bg-griffon">「晚安，我的小电池」</span>
