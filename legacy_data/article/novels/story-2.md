<h1 class = 'HolyUnion'>Scene 01</h1>

> 天堂的倾塌，是从一场肮脏的雪开始的。

汤姆护工最后一次巡逻，他习惯性地看了一眼后墙外那棵老槐树。
树上，落满了黑色的「鸟」。  
它们没有叫，只是静静地看着他，琥珀色的眼睛在黑暗中连成一片，像一场不祥的星空。

冷汗瞬间湿透了后背。  
他立刻将手伸向腰间的左轮，但就在他触碰到枪柄的瞬间——

所有的鸟，在同一时刻，转过了头，用那上百双琥珀色的眼睛，死死地盯住了他。

一股无形的压力瞬间降临，仿佛整个夜空的重量都压在了他的精神上。  
他的大脑一片空白，视野开始扭曲，耳边充满了尖锐的、不存在的嗡鸣。

他想喊，却发现喉咙像是被冰块堵住；  
他想跑，双腿却像灌满了铅。

他所有的注意力，都被树上那片不祥的星空吸走了。

因此，他完全没有察觉到，一个黑色的、高大的人影，已经像影子一样从墙角的黑暗中滲出，  
无声地来到了他的身后。

一只戴着皮手套的手，轻轻地、却又无比有力地捂住了他的口鼻。  
另一只手，一把锋利的短刀，从下至上，精准地捅入了他的心脏，然后用力一绞。

汤姆的身体剧烈地抽搐了一下，然后软倒下去。

夜很深，风织祈被窗外一声异响惊醒。

不是风声，也不是上次的那只野猫。  
那是一种……湿润的、被拖拽的摩擦声，断断续续，正朝着后门的方向远去。  
声音很轻，但在这死寂的夜里，像指甲刮过心脏。

几秒后，摩擦声消失了。  
福利院的呼吸停滞了。

不一会，玻璃破碎声、门被撞开的爆裂声、陌生人的呵斥声混杂在一起，  
像一把脏兮兮的铁锹，将所有睡梦中的孩子粗暴地铲醒。
宁静不是被打破的，而是被瞬间吞噬的。

爆炸的气浪掀翻了一只泰迪熊玩偶，  
棉絮从破洞的肚子里喷涌而出，在硝烟中像一场肮脏的雪。

<span class="bg-cyan">「这边！快！」</span>  
玛丽亚的声音因恐惧而变调，她正奋力将孩子们推向通往地下室的暗门。

在被推着跑过走廊时，风织祈瞥了一眼身侧布满污渍的窗玻璃。  
那玻璃没有映出她的脸，而是倒映着一个剪影。

老人跛着腿的剪影，如同一棵扎根在风暴中的老树，坚定地守在楼梯口。  
一道银线在黑暗中划过——是那把霰弹枪的枪管。  
枪口焰火一闪，一声怒吼在走廊里回荡。  
紧接着，更多的、扭曲的影子向那棵老树扑了过去。

外面传来又一阵枪响，接着是重物倒地的闷响。  
风织祈突然不会呼吸了。

诗集紧贴着她的胸口，书脊上那修补用的胶带，仿佛是老人最后的体温。

孩子们被玛丽亚领着，挤在冰冷的锅炉后面。  
铁锈和煤灰的味道混合着恐惧，浓得化不开。  
玛丽亚用尽全身力气抵住铁门，她的脊背绷成一张绝望的弓。

铁门在沉重的撞击下呻吟。  
透过门底的缝隙，风织祈能看见一双双影子在来回晃动。

她能听见玛丽亚压抑的、带着哭腔的喘息，和撬棍在水泥地上刮擦的、尖锐的悲鸣。  
那是她最后的、徒劳的抵抗。

<span class="bg-griffon">「听听，多卖力啊。」</span>  
声音里满是讥讽，  
<span class="bg-griffon">「可惜，对一群只想要货物的鬣狗来说，眼泪是最没价值的添头。」</span>

门，在一声巨响中向内塌陷。

玛丽亚被门板狠狠撞飞，像个破旧的布娃娃一样摔在角落，没了声息。  
她那沾着油渍的围裙，此刻像一张哭泣的脸。

几个高大的人影涌了进来，他们的轮廓在昏暗中模糊不清，仿佛是从墙壁的阴影里渗透出来的。  
他们没有脸，没有声音，只有一双双伸向孩子们的手。

恐惧像冰水灌入肺里。风织祈的身体在发抖，牙齿在打颤，她和其他孩子一样，被钉在了原地。

一个影子走向了角落。  
但他的目标不是风织祈，而是蜷缩在她旁边、更矮小的艾米。

艾米发出了小猫一样的呜咽，绝望地向后缩去。  
风织祈看着那只巨大的、戴着皮手套的手，即将抓住艾米纤细的胳膊。

_「他们倒下了。」_  
院长的剪影在她脑海中一闪而过。

_「他们都为了保护大家而倒下了。」_  
玛丽亚哭泣的脸也浮现出来。

_「现在，轮到我了吗？」_  
_「可是我……什么都做不了……」_  
这个念头再次浮现，但这一次，它带来的不再是麻痹的恐惧，  
而是一股滚烫的、无法忍受的羞耻。

_「不。」_  
风织祈的颤抖，奇迹般地停止了。

她不再蜷缩。  
她那双死寂的蓝色眼睛里，第一次燃起了某种东西。  
那不是希望，也不是勇气，而是一种被逼到绝境的、野兽般的凶狠。

她的目光飞快地扫过地面，最终锁定在身边一根从墙上被震落的、生锈的铁管上。

她抓住了它，冰冷、粗糙的铁锈硌着她的掌心。

在那个影子即将触碰到艾米的前一秒，风织祈发出一声自己都感到陌生的、嘶哑的尖叫，  
从地上弹起，像一头扑向巨熊的幼狼，用尽全身的力气，将铁管狠狠地砸向了影子的后脑。

**——铛！**  
一声沉闷的金属撞击声。  
影子的动作，停滞了一瞬。

风织祈的心脏几乎要跳出胸腔。  
_「有用吗？我做到了吗？」_

下一秒，那个影子缓缓地、极其不耐烦地转过头。  
他甚至没有去摸自己的后脑，只是用那双藏在阴影下的、毫无感情的眼睛，瞥了她一眼。

那眼神，像在看一只恼人的虫子。

他随意地一挥手。  
巨大的力量传来，风织祈感觉自己像是被一辆全速行驶的车辆撞中，整个人横飞出去，  
后背重重地砸在冰冷的锅炉上，然后滑落在地。

她失败了。

她用尽了自己所有的、能鼓起的勇气，  
用尽了自己全部的力量，发动了人生中第一次反抗。  
结果，只是换来了对方一个轻蔑的眼神。

她躺在地上，视线开始模糊，只能眼睁睁地看着那个影子转过身，再次、毫无阻碍地伸向了艾米。

一切都没有改变。  
她的反抗，她的决心，她的勇气……毫无意义。

这份认知，比身体的剧痛，比死亡的恐惧，要痛苦一万倍。  
那是灵魂被藐视，被碾碎。

憎恨。

前所未有的、纯粹的、足以焚烧整个世界的憎恨，从那破碎的灵魂深处，轰然引爆。  
她不再憎恨入侵者，不再憎恨自己的无力。

她憎恨这个世界。  
憎恨这个让她拼尽全力也无法撼动分毫的、冰冷坚硬的“规则”。

_「……像这样……什么都做不了……」_  
_「……才不要……」_

这时，脑中的那个声音再一次炸响开来。  
它不再是之前的嗤笑或低语，而是一种带着狂喜和赞许的咆哮：

<span class="bg-griffon">「没错！就是这个！这才是你应该说的话！」</span>  
<span class="bg-griffon">「光在心里想有什么用？用你全部的意志！喊出来啊！」</span>

风织祈猛地抬起头，那张沾满血污的小脸上，表情因极致的愤怒而扭曲。  
她死死地盯着那个影子，用尽最后一丝力气，从喉咙的最深处，发出了她对这个世界的、最终的诅咒：

<span class="bg-wind">「去死...<span style="font-size: 115%">去死...</span><span style="font-size: 135%">去死啊!!!</span>」</span>

风织祈身后的阴影，在锅炉的铁壁上疯狂蠕动。  
它不再是二维的影子，而是像沸腾的沥青一样鼓起、剥离。  
一声非人的尖啸，由纯粹的黑暗与电光构成的巨鸟从阴影中爆射而出！

它没有理会其他入侵者，而是以肉眼无法捕捉的速度，径直斯穿了正抓向风织祈的那个影子。  
那人影发出了撕心裂肺的惨叫，倒在地上疯狂抽搐，仿佛灵魂被直接啄了一口。

混乱中，那只猛禽盘旋而下，  
由阴影构成的、介于虚实之间的利爪，一把抓住了风织祈纤细的手臂。  
那触感冰冷，却带着不容置喙的力量。

在其他人影反应过来之前，猛禽猛地振翅，撞破了锅炉房高处的玻璃窗。  
玻璃碎片如雨点般落下，在火光中闪烁着，像是这个破碎天堂最后的眼泪。

风织祈被巨大的力量带向空中，锈城的冷风灌入她的肺里。  
她最后回头看了一眼，

那座曾经是「家」的福利院，此刻正在火光中无声地哀嚎。

而她的同伴，她的守护者，都留在了那片地狱里。

<h1 class = 'HolyUnion'>Scene 02</h1>

> 坠落的终点，是世界的垃圾场。

风织祈被丢进了垃圾堆里。  
柔软的垃圾袋和腐烂的纸箱勉强缓冲了冲击，但那股酸臭、腐败、令人作呕的气味瞬间包裹了她。

那只猛禽，此刻正摇摇晃晃地收拢翅膀，停在旁边一个生锈的集装箱顶上。  
它那由流动的阴影和电光构成的身躯在月光下显得比之前黯淡了许多，仿佛随时会消散。

<span class="bg-griffon">「哎呀，抱歉抱歉。大小姐该不会以为我会把你像羽毛一样轻轻放下吧？」</span>  
它的声音依旧尖酸刻薄，试图用惯常的语气刺激她，但风织祈没有动。

<span class="bg-griffon">「啊啊？感谢我救了你？小事而已不用抱歉啦...」</span>  
她只是蜷缩在垃圾堆里，像一只受伤的小老鼠。  
远方天际线上跳动的火光，像一道无法愈合的伤口。

<span class="bg-griffon">「什么？！要是不报答的话你会过意不去？！」</span>  
跛脚的剪影，倒下的身影，惊恐的脸……  
这些画面在她脑海里反复冲刷，让她失去了思考的能力。

<span class="bg-griffon">「无论如何也要是吗？！值得表扬呢！~」</span>  
<span class="bg-griffon">「你都讲到这个份上了，真是拿你没辙呢——」</span>  
黑色猛禽的独角戏还在继续，  
但它很快发现，往常那些能激起对方一丝波澜的嘲讽，此刻都像石子投入了死海。  
它的声音里，第一次出现了一丝不易察觉的焦虑。

它从集装箱上一跃而下，无声地落在风织祈面前。  
眼睛凑得很近，审视着她空洞的眼神。

<span class="bg-griffon">「那么就...」</span>  
它压低声音，试图用它认为最有效的方式将关系稳定下来，  
<span class="bg-griffon">「来签订契约吧。」</span>

风织祈缓缓地、非常缓慢地抬起头。  
脸上沾着污渍，银色的发丝上挂着不知名的黏腻液体。  
她的眼睛仿佛隔着一层毛玻璃在看这个世界。  
涣散、没有焦点，

她张了张嘴，喉咙里发出的不是语言，而是一声微弱的、破碎的气音。

然后，她问出了那个在她心中唯一盘旋的问题，那个比“契约”和“生存”都重要一万倍的问题。  
声音沙哑、干涩，几乎听不见：  
<span class="bg-wind">「……为什么……不救他们？」</span>

显然它愣住了。  
但随后并没有嗤笑，也没有暴躁地解释。  
它爆发出的是一种被至亲背叛般的、歇斯底里的**暴怒**。

<span class="bg-griffon">「他们？他们他们他们！你的脑子里就只有那群死人吗？！」</span>
它的声音不再是戏剧性的尖利，而是因为愤怒而变得沙哑和扭曲，像金属摩擦。  
它不再像是魔物，而是像一个愤怒的灵魂。

<span class="bg-griffon">「你看看你现在这副德行！为了那群已经没救的家伙，你想死在这里吗？！」</span>

它猛地抬起由阴影构成的翅膀，**狠狠地**、**抽了风织祈一巴掌**。  
清脆的响声在垃圾场里显得格外突兀。

风织祈被打得偏过头，嘴角渗出一缕血丝。  
但她的眼神依旧是涣散的。

<span class="bg-griffon">「我救了谁？我救了「你」！我从那片火海里，把「我们」给拖了出来！而你现在却在问我，为什么没有回去陪葬？！」</span>

它凑得极近，琥珀色的眼睛里燃烧着纯粹的怒火，几乎要灼伤风织祈的脸。

<span class="bg-griffon">「回答我！那个老头，那个女人，那些小鬼……他们拿命换你活下来，是想看到你像条蛆一样，烂死在这个垃圾堆里吗？！」</span>

每一个字，都像一把烧红的刀子，捅进风织祈的耳朵。  
它的话语，绕开了所有逻辑和道理，直击她内心最深处的。

风织祈的身体开始微微颤抖。  
她无法回答。  
因为她知道，这家伙说的，**是对的**。

这份正确，比任何辱骂和亵渎都更残忍。  
它让她意识到，她连沉溺于悲伤的资格都没有。

因为她的“生”，是建立在他们的“死”之上的。  
她背负着“活下去”的、最沉重的责任。

风织祈不再看它，也不再说话。

她无法再恨这家伙了。  
因为它所做的，正是守护者们希望的——让她活下去。  
她也无法再恨自己了。

她什么都无法再感受了。  
只是缓缓地、缓缓地松开了手。

那本一直被她紧紧抱在怀里、仿佛是她与过去唯一连接的诗集，从她怀中滑落。  
「啪」的一声，掉进了脚下的泥水里。

书页迅速被肮脏的液体浸透、染黑。

<span class="text-deep-orange">我却看见那儿尽是坟墓，</span>  
<span class="text-deep-orange">墓碑代替了原有的花朵：</span>  
<span class="text-deep-orange">穿黑袍的牧师还在踱来踱去，</span>  
<span class="text-deep-orange">用荆棘捆住了我的欲念和欢乐。</span>

风织只是看着，脸上没有任何表情。  
这不是一次放弃，而是一次“承认”。  
承认自己无法再面对这份沉重的“生存”的责任。

那双蓝色的眼睛里，最后一丝光亮熄灭了。

黑色猛禽的咆哮戛然而止。  
它赢了道理，却输掉了情感。  
它用最正确的话，把她逼入了死局。

它突然感觉到一阵前所未有的虚弱，仿佛自己与这个世界的连接正在被切断。  
它的身体开始变得不稳定，边缘像蜡烛一样融化、剥离。

这一次，它的声音里不再有愤怒，只剩下一种真正的、被抛弃的恐慌。

<span class="bg-griffon">「喂！你……你干什么？！」</span>  
它猛地向前一步，声音因为恐惧而变得尖利，  
<span class="bg-griffon">「是我错了行不行！我说错话了！别这样……想点什么……随便什么都好……求你了……别丢下我……」</span>

它在哀求。  
哀求“自己”，不要杀死“自己”。

但太晚了。

在女孩那死寂的注视下，它发出一声不甘的、被拉长的悲鸣。  
最终彻底溃散成一缕墨迹，融入了地上的泥水之中。

垃圾场重归寂静。

锈城的第一缕晨光，穿透云层，照亮了垃圾场的顶端。  
风织祈缓缓地站起身，像一个被重新上紧发条的提线木偶。

她低头，看了一眼脚下那本浸泡在泥水里的诗集，脸上没有任何表情。  
她弯下腰，机械地将它捡起。

她用已经脏污不堪的袖子，一遍又一遍地擦拭着封面上的泥泞。  
这个动作，并非出于珍视，而更像是一种被铭刻在身体里的、无法磨灭的“习惯”。  
就像呼吸，就像心跳。  
即使“心”已经死了，但身体的本能依然在运作。

然后，她转过身，抱着那本冰冷而湿透的书，一步一步地，走进了锈城更深的、永不天亮的阴影之中。
