<h1 class = 'HolyUnion'>Scene 01</h1>

> 麻木是伤口结的痂，但总有人会把它撕开。

杂物间的灯泡接触不良，在头顶发出滋滋的电流声。  
昏黄的光线像垂死者的呼吸般明灭不定，将瘦小的影子投在发霉的墙纸上，时而拉长时而扭曲。

她蜷缩在堆积的米袋后面，膝盖抵着胸口，那本发皱的诗集摊开在她面前的旧报纸上。  
指尖轻轻划过那些被反复翻阅而卷边的书页，这是她每日的仪式。

水龙头没拧紧，后厨传来滴答、滴答的水声，像一颗永远走不完的秒针。

突然——

**砰！**

门被猛地推开，堆积的空罐头盒哗啦一声倒塌。  
灰尘从天花板簌簌落下，刺眼的灯光像刀一样劈进黑暗。  
风织浑身一颤，下意识地将书合上，护在怀里。

<span class="bg-gray">「哈……原来小哑巴平时躲在这儿？」</span>

带着酒气的嘟囔声渗进来，站在门口的人影摇摇晃晃  
——是老板的儿子，那张浮肿的脸上泛着酒精催生的潮红，左眼下方有道新鲜的擦伤正在渗血。

他的目光像黏腻的蛞蝓般扫过杂物间，最终钉在米袋后的银发上。  
他摇摇晃晃地往前一步，皮鞋踩在潮湿的地板上，发出恶心的声响。

风织没有动，甚至连呼吸都停滞了一瞬。

啤酒瓶被他随手扔到墙角，玻璃碎裂的声音让风织的肩膀剧烈颤抖。  
她本能地将书往身后藏的动作，引起了对方的注意。  
醉汉咧开嘴笑了，露出参差不齐的黄牙:  
<span class="bg-gray">「藏什么好东西呢？给老子看看！」</span>

他伸手去抓她的书。

风织的反应比她自己预想的还要快  
——她猛地往后缩，后背撞上冰冷的墙壁，书本紧紧护在胸前。

这个抗拒的动作，似乎取悦了他。  
他一把抢过那本破旧的书，胡乱地翻着，像在翻一本毫无价值的废品目录。

风织的眼神瞬间变了。  
那不是愤怒，也不是恐惧，而是一种……类似于程序被错误指令打断后，系统陷入死寂的空白。

他眯着醉眼，装模作样地念出上面的诗句：  
<span class="bg-gray">「『他……听见羔羊……天真地呼唤』？哈！什么玩意儿！」</span>

他的口水，溅在了泛黄的书页上。

那一刻，风织的世界里，某个至关重要的“规则”被打破了。  
这个男人，这个肮脏的、充满酒气的存在，正在「污染」她的「圣物」。  
这是她麻木世界里唯一洁净、有序的东西。

她缓缓抬起头，那双死寂了已久的蓝色眼睛里，没有燃起火苗，  
反而变得像冰一样，绝对的、零度的冰。  
她的嘴唇动了动，喉咙里挤出了嘶哑、干涩，却异常清晰的两个字。

<span class="bg-wind">「放开。」</span>

这不是请求，也不是警告，而是一个陈述句。  
就像在说「把东西放回原处」。

醉汉明显愣住了，但随即爆发出更兴奋的大笑：  
<span class="bg-gray">「操！原来会说话啊！」</span>  
他把书狠狠摔在地上，用脚踩了上去，碾了碾。  
<span class="bg-gray">「再多说几句来听听？」</span>

风织没有再说话。  
她的内在世界，那座由重复和仪式搭建的精密钟楼，已经彻底崩塌了。  
现在，驱动她的只剩下一个最底层的、冰冷的念头。

_「……错了。」_  
_「……要把它放回去。」_  
_「……要把它擦干净。」_

她无视了那个男人，径直朝地上的书伸出手去。

这个动作彻底激怒了醉汉。  
他一把拽住风织的头发将她拖到房间中央，随后扯住她胸口的衣服猛地把人往上拽起来。

**嘶啦——**

早已脆弱的布料应声撕裂。  
风织重重摔在地上，头晕目眩。

她没有在意。  
她的眼睛，依然死死地盯着不远处地板上的那本书。  
她挣扎着想爬过去。

<span class="bg-gray">「妈的……还真干净……」</span>  
醉汉的眼神变了，从单纯的施虐，变成了赤裸裸的欲望。  
他压了上来，粗糙的手掌掐着她的腰，将她死死地按在地上。

风织的动作停住了。  
不是因为恐惧，也不是因为羞耻。  
而是因为，她动不了了。

一个巨大的、温热的、散发着恶臭的「东西」，压住了她，阻止了她去完成那个必须完成的「任务」。

她能感觉到对方在她身上摸索，能感觉到他粗重的喘息，能感觉到他黏腻的口水滴在她的皮肤上。  
但这些，对她来说，都只是遥远的、没有意义的背景噪音。  
就像苍蝇的嗡嗡声，就像水龙头的滴答声。

她唯一能感知到的，是那个越来越远的目标  
——那本躺在地上、被踩上脚印的书。

一个念头，像冰块一样在她空白的脑海中成型。

_「……碰不到。」_  
_「……这个东西，挡住了。」_  
_「……必须……把它移开。」_

她开始在黑暗中无声地数数。  
_一、二、三……_  
这不是在对抗地狱。  
这是在思考。

用一种她已经很久没有使用过的、绝对冷静的方式，思考着如何“移开”这个“东西”。

数到十一的时候，她的指尖碰到了地板上的某样东西——  
一块碎玻璃。  
是被他砸碎的酒瓶，边缘锋利、触感冰凉、像一块死去的星星碎片。

思考，结束了。

_「……可以了。」_

<h1 class = 'HolyUnion'>Scene 02</h1>

> 当祈祷无效时，双手会自己寻找答案。

数到十一的时候，她的指尖碰到了地板上的某样东西——  
一块碎玻璃。  
是被他砸碎的酒瓶，边缘锋利。触感冰凉，像一块死去的星星碎片。

风织的手指收紧了。

他还在她身上喘息，粗糙的手掌掐着她的腰，另一只手忙着解开自己的裤子。  
他根本没注意到——被他按在身下的小哑巴，眼神已经变了。  
蓝色的瞳孔，像是冰层下的湖水，突然冻结。

**咔！**

玻璃碎片刺进眼球的触感很奇妙  
——先是遇到一层柔软的阻力，然后「噗」地一声穿透了那层薄膜。  
温热的液体溅在她的手指上，黏腻得像融化的糖浆。

它的嚎叫声几乎掀翻屋顶。

风织缓缓坐起身。  
她的动作很慢，像是被某种无形的线牵引着。

她站了起来。  
膝盖在发抖，但身体却像被某种陌生的力量支配着——冰冷、精确、毫无感情。  
她低头看着自己沾满血的手，然后慢慢走向后厨的方向。

剁骨刀就挂在墙上。  
握在手里时，沉甸甸的质感让她想起了院长修东西用的铁锤。

它还在地上痛苦地翻滚，挣扎着想要爬起来。  
<span class="bg-gray">「等……等等……！」</span>  
它终于意识到了什么，声音开始发抖：

<span class = 'bg-gray'>「我错了……我不该……！」</span>

风织没有回答。  
只是举起刀然后劈了下去。

_「哦？终于不当哑巴了？我还以为你打算被干到死，都不吭一声呢。」_

刀刃咬进皮肉，它的喉咙发出脆响。  
_「瞄准喉咙？想让他也当个『哑巴』吗？就像你这三个月一样？」_

这次是横向劈砍，肌腱断裂。  
_「对，就是这样！多好的教材啊，大小姐。」_

砍进去时遇到柔软的阻力，肠子蠕动着缠住刀刃。  
_「看看这些东西，是不是和玛丽亚给你炖的土豆一个颜色？哦，不对，那个更软烂一点。」_

她搅动刀柄时，一段结肠从切口滑出。  
_「哎呀，掉出来了。要不要捡起来，像你捡那本被你扔掉的书一样？」_

她停了下来，已经记不清自己砍了多少下。  
手臂机械地抬起、落下、抬起、落下……  
骨头碎裂的声音、肌肉被撕裂的声音、内脏被挤压的声音……

脑海里那个喋喋不休的声音，也随着她动作的停止而安静下来，只留下一声心满意足的、轻微的饱嗝声。  
剁骨刀的刃口已经卷了边，沾满了碎肉和骨渣。

她瘫坐在地上。  
手臂的剧痛、肌肉的酸涩、空气中浓得化不开的血腥味、皮肤上黏腻的汗水和血水……  
这些被她隔绝了三个月之久的、尖锐的“感觉”，像决堤的洪水一样，冲垮了她用麻木筑起的堤坝，蛮横地灌满了她的全身。

**她活过来了。**  
**以一种最痛苦的方式。**

她开始哭。  
不是抽泣，不是呜咽。  
是无声的崩溃。  
是身体在为这场残酷的“复苏”而痉挛。

只有眼泪大颗大颗地往下掉，冲刷着脸上的血迹，留下蜿蜒的痕迹，一滴、两滴……落在血泊里消失不见。

最后目光洒落在地上摊开的诗集：  
那一页上印着一个肮脏的脚印。

<span class="text-deep-orange">因此他就展翅飞走：</span>  
<span class="text-deep-orange">清晨抹上玫瑰红的娇羞</span>  
<span class="text-deep-orange">我擦干了眼泪，开始武装，</span>  
<span class="text-deep-orange">用一万副盾牌与长矛使我不心慌。</span>
