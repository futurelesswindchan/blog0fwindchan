<h1 class = 'HolyUnion'>Scene 01</h1>

> 锈城的黄昏像生锈的刀刃泛着回光。

穿过爬满藤蔓的铁栅栏，  
霓虹广告牌的紫光从福利院老旧的玻璃窗渗进来，将整条走廊腌制成褪色的泡菜坛子。

在这片紫光照不到的角落阴影里，风织祈正蜷缩着，银发垂落，遮住半张脸。  
她总觉得阴影比光亮处更安心，仿佛有什么东西在黑暗中与她一同呼吸。

手里捧着的玻璃糖罐是玛丽亚三天前给的。  
里面还剩四颗薄荷糖。

罐体折射出的光斑在她手背游移，像被囚禁的萤火虫。

扫帚扫地的沙沙声停在了风织祈身边。  
一只指节粗大皲裂，像老树根一样的手，轻轻放在了她的头顶。  
但那掌心是暖的。

<span class="bg-cyan">「小星星，」</span>  
玛丽亚护工的声音也像她的手一样，带着一丝粗糙的沙哑，却很温柔，  
<span class="bg-cyan">「躲在这里，会着凉的。」</span>

风织祈往走来的玛利亚怀里钻了钻。  
她总是这么称呼她，尽管册子上写的是`「灰羽风织祈」`。

五个孩子正在走廊尽头跳房子，  
格子是用粉笔画在龟裂地砖上的。  
嬉笑声撞在生锈的管道上，发出空洞的回响。

玛丽亚也笑着走过去，如变魔术般从袖口摸出一堆水果糖，递放在他们掌心。  
<span class="bg-cyan">「今天有草莓味的哦。」</span>

风织祈的耳膜深处，却似乎有一个更尖锐的声音在低语，  
像是不耐烦的嗤笑，将那些快乐的声音撕得粉碎。

她不明白这嗤笑从何而来，就像不明白自己为什么总爱躲在影子里一样。

维克托院长从走廊尽头走来，跛着的腿让他每一步都像在丈量着痛苦。  
他左手紧紧抱着一本用绝缘胶带缠着书脊的旧诗集，仿佛那是他另一根拐杖。

院长的右手缠着绷带，准是不小心被装订机夹伤了。

他在风织祈面前停下。

<span class="bg-amber">「第8页补好了。」</span>  
老人把书轻轻放在风织祈膝头，声音低沉而平静。  
身上的陈年烟草味混着薄荷糖气息笼罩下来。

风织祈翻开泛黄的书页。

新补的段落夹着张糖纸作为书签，折成歪扭的星星形状。

走廊另一头，几个大孩子为抢夺电视遥控器而吵嚷起来，老旧动画片的嘈杂配乐充满了整个空间。  
他走过去，拿起遥控器关掉了电视。屏幕一黑，仿佛将外界的喧嚣隔绝在外。

他没有回头，只是用平静的声音说：  
<span class="bg-amber">「好了，动画片时间结束了。谁想听我讲个关于会飞的狮子的故事？」</span>

吵闹声平息了，孩子们围了过来，期待地看着他。

就在这时，福利院的大门被吱呀一声推开，带着一阵傍晚的凉风。

<span class="bg-deep-purple">「回来啦——！看看我带了什么好东西！」</span>

汤姆护工提着两个沉甸甸的袋子走进来。  
他的脸被风吹得有些红，但笑容比霓虹灯还要明亮。  
他献宝似的从袋子里掏出一大块还带着热气的奶油面包，整个走廊瞬间被麦子的香气填满了。

<span class="bg-gray">「哇——！」 </span>  
孩子们立刻抛弃了“会飞的狮子”，欢呼着朝汤姆冲了过去。

<span class="bg-deep-purple">「别急，别急，人人有份！」</span>  
汤姆笑着，将孩子们护在怀里。

<span class="bg-cyan">「汤姆！跟你说了多少次，不要给他们吃这么多甜点啦！晚饭又吃不下了！」</span>  
玛丽亚嘴上抱怨着，却已经转身去厨房拿刀和盘子了。

汤姆冲维克托院长挤了挤眼睛，压低声音，像个共谋者一样说：  
<span class="bg-deep-purple">「院长，我还偷偷买了您最喜欢的那种烟草。」</span>

维克托的脸，线条似乎柔和了一点，他哼了一声，算是回应。

风织祈看着这片热闹的景象，  
面包的香气、孩子们的笑声、玛丽亚的脚步和汤姆爽朗的声音交织在一起。  
她下意识地，将怀里的糖罐与诗集抱得更紧了一些。

汤姆特意走到了角落，撕下一块柔软的面包，笨拙却不容置疑地塞进风织祈手里。  
而风织祈，在犹豫之后，小口地吃掉了。

维克托院长重新在孩子们身旁坐下，低沉的声音混在背景的喧闹中，显得格外沉稳：  
<span class="bg-amber">「好了。现在，我们可以讲那个关于会飞的狮子的故事了。」</span>

走廊的灯泡忽然闪烁了一下，昏黄的光线短暂地照亮了放在风织祈膝头的书。  
新补的那页上写着：

<span class="text-deep-orange">他听见羔羊天真地呼唤</span>  
<span class="text-deep-orange">他听见母羊温柔地答腔</span>  
<span class="text-deep-orange">他小心看守，它们也平安</span>  
<span class="text-deep-orange">因为它们晓得牧童就在近旁</span>

底下贴着张泛黄的照片：  
去年的全员合影，维克托院长站在最左边，右手搭在了一个孩子的肩膀上。  
那个被院长搭着肩膀的、面无表情但也没有躲开的孩子，就是去年的她。

风织祈把糖罐举向头顶，灯光透过糖罐，变成彩色的光斑。  
光斑掠过合影上玛丽亚的笑脸，扫过汤姆缺了门牙的笑容，  
最终，停在了小艾米头顶的蝴蝶结上。  
那个总爱送给她花的五岁女孩。

风织祈低下头，合上书，把它紧紧抱在怀里。  
玻璃糖罐里的光芒依然温柔地折射着，像是某种虚假的承诺。

仿佛只要躲在这片彩色的光晕里，外面的世界就永远不会闯进来。

<h1 class = 'HolyUnion'>Scene 02</h1>

> 深夜的福利院像艘正在漏水的旧船。

风织祈本该睡着了。  
至少大人们是这么以为的。

光脚踩在走廊地砖上，寒气顺着脚趾爬上脊椎。  
书贴着衣服下摆，修补的胶带被体温焐得微微发粘。

厨房门缝漏出的烛光在地板上切开一道伤口，  
里面传来压低的争执声。

<span class="bg-deep-purple">「他们又来了。就在后门，我看见了……影子，很多影子……在那里……」</span>  
汤姆护工的声音像砂纸擦过铁锈。  
<span class="bg-deep-purple">「刚刚……我听到后门的门栓有被拉动的声音。很轻，但我听见了。」</span>

老维克托没有说话，只是从橱柜深处拖出个大铁盒。  
生锈铰链的呻吟声让风织祈想起福利院大门的声响。

盒子里躺着一把保养良好的泵动式霰弹枪，油光在枪管上流动如蛇。  
院长往弹巢填子弹的动作娴熟得令人心寒。

他将霰弹枪靠在桌边，然后从腰包里，拿出了那把老式左轮手枪，推到了汤姆面前。  
汤姆看着桌上的枪，喉结上下滚动了一下，伸出的手有些颤抖。

玛丽亚捂住嘴的指缝漏出半声呜咽。  
她围裙上还沾着一丁点做晚餐削掉的土豆皮，此刻正随着颤抖的肩膀簌簌掉落。  
<span class="bg-cyan">「听说上周东区收容所……那边的孩子们都……」</span>

维克托将最后一颗霰弹填满，拉上了枪栓，发出“咔哒”一声清脆的金属合拢声。  
他抬起头，眼白在烛火下泛着浑浊的黄。他没有看玛丽亚，而是盯着汤姆，声音沙哑但异常镇定。

<span class="bg-amber">「汤姆，你守后半夜。有任何不对劲，先对天开枪示警，然后退到地下室。」</span>  
他顿了顿，目光转向玛丽亚，语气放缓了一些，但依然不容置喙。  
<span class="bg-amber">「玛丽亚，明天开始，把地下室的杂物清出来。</br>我们要……演练一下紧急避难的流程。」</span>

他拿起桌上已经冷掉的烟斗，又放下，似乎在寻找一个合适的借口。  
<span class="bg-amber">「就跟孩子们说……我们在玩一个新的游戏，叫‘躲猫猫大赛’。</br>谁能最快地跑到地下室并且保持安静，谁就能得到奖励。就这么说。」</span>

风织祈的脚趾无意识抠着地砖裂缝。

走廊尽头传来梦呓，某个孩子正在哭喊「妈妈」，  
尽管这里所有人都是被遗弃的星星。

厨房里的交谈声戛然而止。

并非因为他们达成了共识，而是因为窗外传来了一声轻微的、仿佛指甲刮过玻璃的异响。  
维克托的动作停住了，他缓缓抬起头，死死地盯住了厨房那扇高窗。

一只黑色的猛禽，不知何时落在了窗外的铁栏上。  
它没有叫，只是歪着头，用那双不似活物的、琥珀色的眼睛，静静地、一动不动地凝视着厨房内的烛火。

汤姆的呼吸声变得粗重，他下意识地后退了一步，撞到了身后的橱柜。

维克托没缓缓举起了手中的霰弹枪。  
但他的枪口，并没有对准那只鸟。  
而是稳稳地、决绝地，对准了厨房通往走廊的——门。

仿佛在说，无论窗外有什么在窥伺，他要守护的，是门内的这个家。

厨房里的烛火剧烈摇晃起来。  
风织祈看见维克托起身时投在墙上的影子：  
一个佝偻的老人，一杆笔直的枪。  
影子随着烛光扭曲拉长，像头蓄势待发的兽。

当风织祈退回阴影时，怀里的诗集硌得肋骨生疼。

<span class="text-deep-orange">我们的嬉戏就此收场：</span>  
<span class="text-deep-orange">有多少兄弟姊妹，</span>  
<span class="text-deep-orange">依偎在妈妈周围，</span>  
<span class="text-deep-orange">就像巢里的小鸟，</span>  
<span class="text-deep-orange">准备着要睡觉了：</span>  
<span class="text-deep-orange">再也看不见有人嬉戏，</span>  
<span class="text-deep-orange">在那渐渐暗下来的草地。</span>

她躺在床上，用被子把自己裹成一个茧。  
但寒意并非来自窗外，而是来自骨髓深处。

闭上眼，那只黑鸟停留的画面就无限放大，

一个念头突兀地闯入脑海，带着一种不属于她的、充满暴戾的饥渴：

<span class="bg-griffon">「——这可不够。」</span>

风织祈猛地睁开眼，死死盯住天花板，直到那陌生的饥饿感像潮水般退去。
