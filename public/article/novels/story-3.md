<h1 class = 'HolyUnion'>Scene 01</h1>

> 当一个世界死去，幸存者就成了它的幽灵。

雨，带着铁锈和机油的混合气味，冲刷着锈城。
风织在巷子里行走。

她不记得自己走了多久，也不在乎要去哪里。  
世界被简化成一片模糊的、流动的霓虹光斑，和砸在脸上冰冷的雨水。  
怀里那本湿透的书，像一块浸满悲伤的海绵，沉重得让她几乎直不起腰。

腹中传来一阵剧烈的绞痛，那是饥饿在啃食她的内脏。  
她踉跄了一下，扶住满是涂鸦的墙壁，剧烈地干呕起来，却什么也吐不出。

寒意从湿透的衣物渗透进皮肤，她开始无法抑制地发抖。  
视野边缘开始发黑，霓虹的光斑在黑暗中拉长、旋转，像一个巨大的、嘲弄的漩涡。

她要死了吗？

这个念头平静得像是在陈述别人的事。

脑海深处，仿佛有一个极其微弱的声音在叹气：  
_「……会死的……」_

是谁在说话？  
她想不起来。也懒得去想。

她靠着墙，又站了一会儿，然后继续向前走。

不知过了多久，她终于再也支撑不住，膝盖一软，沿着墙壁滑倒在地。  
雨水的气息混合着泥土的腥味，钻入她的鼻腔。

她侧躺在垃圾堆旁，蜷缩起身体，像一只被抛弃的、即将死去的野犬。

雨还在下。

意识的最后，她看到一扇后门被推开。  
一个叼着廉价香烟的男人提着垃圾桶走了出来，昏黄的灯光从门里漏出，在他脚下投出一小片干燥的、温暖的方形。

他将垃圾倒掉，金属桶与地面碰撞发出刺耳的声响。  
他似乎注意到了蜷缩在角落的阴影，皱着眉走了过来。

男人吐出一口烟，烟雾在雨中迅速散开。  
他用沾着油污的皮鞋尖，不轻不重地踢了踢风织的后背。

<span class="bg-gray">「喂。」</span>

没有回应。

他蹲下身，借着门里透出的光，粗暴地拨开她黏在脸上的银发。  
当他看清那张毫无血色、却异常干净的小脸时，眼神微微变了一下。

他伸出两根手指，探了探她的鼻息。

还有气。

男人站起身，又抽了一口烟，目光在她瘦小的身体上刮来刮去，  
像是在评估一头走失的牲畜，计算着它还能产出多少价值，需要多少饲料。

他看着她怀里那本被死死抱住的、湿透了的书，眼神里闪过一丝不耐烦。

最终，他掐灭了烟头，扔进水洼里，发出「滋」的一声。  
他没有再多看一眼，转身走回了餐馆。

就在风织的意识即将彻底沉入黑暗时，那扇门又被打开了。  
男人去而复返，一言不发地抓住她的一只胳膊，像拖一个麻袋一样，将她拖进了那片方形的光亮之中。

门，在他们身后关上了。  
隔绝了雨，也隔绝了整个世界。

<h1 class = 'HolyUnion'>Scene 02</h1>

> 当生命只剩下重复，仪式便成了唯一的意义。

餐馆的后厨，没有白天黑夜，只有一盏永远亮着的、蒙着油污的昏黄灯泡。

天还没亮，风织已经站在了那口巨大的水槽前。  
她面无表情，动作机械。  
冰冷的水流冲刷着堆积如山的碗碟，也冲刷着她那双早已被清洁剂泡得发白、起皱的手。

这几个月里，她的世界都被压缩在这间油腻的后厨和漏水的杂物间里。

擦地，拖把的木柄已经开裂，每一次用力都会有细小的木刺扎进掌心。  
倒垃圾，将混杂着馊水和残渣的垃圾桶拖进后巷，那里的气味曾让她呕吐，如今却已习以为常。  
吃饭，等所有人都吃完后，老板会像打发野猫一样，将一盘剩饭剩菜推到她面前。

她不说话，也不看任何人。  
老板很快就发现，这个阴沉的银发小鬼会吓跑客人，于是她被彻底禁止进入前厅。  
她的存在，变成了一个只在后厨活动的、无声的幽灵。

水槽边挂着一面裂开的镜子，她偶尔会瞥见自己的倒影：  
一个瘦小的影子，蓝色的眼睛在昏暗的灯光下显得格外大，像是两颗被遗忘在角落、蒙上灰尘的玻璃珠。  
她从不看第二眼。

一天晚上，前厅的生意格外喧闹。  
隔着那扇油腻的、永远关不紧的门，醉汉们的吹嘘和争吵像潮水一样涌进来。  
风织照例在水槽前洗着碗，对这些噪音充耳不闻。

突然，几个词飘进了她的耳朵，像几片滚烫的铁屑，落在了她冰冷的神经上。

<span class="bg-gray">「……就北区那个福利院，烧得真他妈惨……」</span>  
一个粗哑的男声说，听起来像是喝了不少酒。  
<span class="bg-gray">「听说了吗？官方的说法是锅炉爆炸，意外事故。」</span>

<span class="bg-gray">「屁的意外！」</span>  
另一个声音立刻反驳道，  
<span class="bg-gray">「里面的人，没一个是烧死的！全是……」</span>  
声音压低了，但依然能听清。

风织的动作，停顿了一瞬。  
只有一瞬。

然后，她继续拿起一只油腻的盘子，放进水里。  
但她的耳朵，却不受控制地竖了起来。

<span class="bg-gray">「听说那老院长是个硬骨头，退伍兵，可惜了……还有那些孩子，一个都没跑出来……」</span>  
<span class="bg-gray">「造孽啊……」</span>

一声叹息，很快被新的劝酒声和骰子碰撞的声响淹没了。  
话题转向了哪个舞女的屁股更翘。  
福利院的悲剧，在他们的世界里，只停留了不到一分钟。

风织面无表情地继续洗着碗。  
但水槽里，一只被她握得过紧的玻璃杯，悄无声息地裂开了一道缝。  
锋利的边缘割破了她的掌心，一缕鲜红的血丝混入油污的泡沫中，很快消失不见。

她仿佛没有感觉到疼痛，只是换了个杯子，继续着她的工作。

深夜，当最后的醉汉被赶走，餐馆终于陷入死寂。  
这是唯一属于她的时间。

她会偷偷打一盆清水，躲在杂物间的角落里，用一块洗得褪色的抹布，一遍又一遍地擦拭自己的身体。  
她讨厌油污，讨厌那股仿佛已经渗入骨髓的油烟味。  
仿佛只要保持身体的洁净，就能证明自己还没有和周围的污秽融为一体。

洗漱完毕，她会将那件破旧的衣服仔细洗净、拧干，挂在米袋后面。  
然后，她会像进行某种神圣的仪式一样，蹲下身，掀开那床扎人的、填充着旧报纸的麻袋被子。

——至少那本书还在那里。

封面在坠落和雨水中已经磨损，但院长用绝缘胶带修补的书脊，依然牢固地支撑着它。  
风织小心翼翼地将它捧出来，像是捧着一件易碎的珍宝。

她翻开书页，指尖轻轻抚过院长用香烟盒纸补上的段落，抚过那粗糙但工整的字迹，  
仿佛这样就能触碰到那个早已化为灰烬的世界，感受到那双缠着绷带的手最后的温度。

她不会去读上面的文字。  
因为文字会让她回忆，而回忆是痛苦的。  
今晚，这份痛苦变得前所未有的清晰。

她只是抱着它，将书紧紧贴在胸口，蜷缩在那两块木板拼成的「床」上。

书脊上胶带的触感，是这九十天里，她唯一能感受到的、不属于这个餐馆的「真实」。

<span class="text-deep-orange">我总是在哭泣，夜晚和白天</span>  
<span class="text-deep-orange">他总是把我的泪水擦干，</span>  
<span class="text-deep-orange">我总是在哭泣，白天和夜晚</span>  
<span class="text-deep-orange">我心中的欢悦却向他隐瞒。</span>

她抱着这份唯一的真实，闭上眼睛，强迫自己沉入睡眠。  
……进入一个没有油烟，也没有争吵的世界。  
……一个，或许还会有星星的，虚假的世界。
