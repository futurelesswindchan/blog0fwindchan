## 1. 灵魂拷问：我到底为什么要受这个罪？！

当整个风风博客的代码量突破上万行，当为了一个小小的动画调了一整晚 CSS，小风酱终于在一个深夜停下了敲击键盘的手，陷入了深深的沉思。

**Q: 为什么写这个博客？**

## 2. 造轮子的诱惑：卡吧与图吧的赛博启示录

**Q: 为什么要自己造轮子？明明市面上有那么多现成的博客系统！**  
**A:** 不知道 QAQ！这就好像装机界的那个千古传说：

> 三千预算进卡吧，加钱加到十万八。十核E7装上去，四路泰坦抱回家。
> 4K屏幕组三屏，万元液冷温度压。机械硬盘装三块，固态内存使劲加。
> 键鼠必花几千元，耳机手柄八千八。还有机箱六千元，红星炸弹啪啪啪。
> **最后发现没啥用，抱着神机刷贴吧。**

写风风博客也是一样啊！本来只是想写个记录日常的静态页面，结果上了 Vue3，加了 TS，接着又觉得没有打字机特效不够炫，没有目录小桌宠不够萌…… 最后硬生生造出了一台四路泰坦级别的博客系统。

如果真的是那种专注于技术分享的硬核大佬，估计早就用 VuePress 或者 Hexo 啦！就像垃圾佬一样：

> 三千预算进图吧，一吨配置抱回家。四核至强装上去，老旧主板笑哈哈。
> 蓝光电影皆不惧，青鸟散热是最佳。最多一块破金胜，专用内存使劲加。
> 键鼠一共九块九，外设包邮八块八。还是萝卜散热好，工包电源顶呱呱。
> **最后发现剩太多，学校对面开网吧。**

但这就是折腾的乐趣呀！我们追求的从来不是“能不能用”，而是“这玩意儿是不是我亲手一行行搓出来的”！( •̀ ω •́ )y

## 3. 进化的阵痛：从纯前端到全栈的妥协

**Q: 为什么一开始非要搞纯前端，后来又老老实实改成标准的前后端分离了？**
**A:** 因为手改 JSON 上传文章真的**太！痛！苦！了！** QAQ

想象一下这个反人类的流程：

1. 在本地打开 VS Code 写 Markdown。
2. 极其卑微地打开 `index.json`，手动加上一行 `{"id": "new-post", "title": "..."}`。
3. 敲下 `npm run build`，听着电脑风扇狂转。
4. 把打包好的 `dist` 文件夹拖到服务器。
5. `systemctl restart nginx`。
6. 发现文章里有个错别字！ **回到第一步重新来过！！！** (崩溃抓头发)

所以，为了能够优雅地、随时随地地在网页上修改错别字，小风酱终于向这个世界妥协，含泪学了 Flask，写出了现在这个带 CMS 后台的全栈完全体 OAO。

## 4. 终极的沉默：荒诞与反抗

最后，也是最直击灵魂的一个问题。

**Q: 为啥还要写这些根本没人看的文章 awa？**

... (沉默 ing)

是啊，在这个流量碎片化、短视频满天飞、大家都习惯了刷十几秒的短视频，长图文都没人耐心看完的时代，谁会来看一个不知名小站里的长篇大论呢？  
谁会关心你用了什么算法去渲染瀑布流，谁又会在意那个会晕车的目录小宠物呢？

看着后台少得可怜（甚至可能全是自己和爬虫）的访问量，有时候也会觉得挺好笑的。

但是， **也许在荒诞的世界里，反抗便是我们存在过的证明吧。**

我们在这个浩瀚无垠、却又日益封闭的互联网荒原上，用代码一砖一瓦地搭建起属于自己的小木屋。

- 那个会榨干显卡性能的玻璃面板，是我们的落地窗；
- 那个永远陪伴在右下角的小精灵，是我们的宠物；
- 那些长篇大论的技术连载，是我们刻在木屋墙壁上的成长日记。

就算全世界都没有人路过这里，就算服务器的日志里只有孤独的 Ping 响应，但只要你输入那个熟悉的域名，看到页面如约亮起，看到那些属于你的粒子在屏幕上跳跃——

**那一刻，你就拥有了整个赛博宇宙。** 🌌

或许这就是西西弗斯推石头的意义吧。

哪怕没有观众，我们也要把舞台搭建得无比华丽；哪怕无人喝彩，代码里那些俏皮的注释也是我们留给这个世界的浪漫。
